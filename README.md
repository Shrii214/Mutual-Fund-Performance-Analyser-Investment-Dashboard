# Mutual-Fund-Performance-Analyser-Investment-Dashboard
End-to-End Mutual Fund Performance Analysis and Investment Dashboard using Python, SQL, Excel, and Power BI featuring data cleaning, EDA, fund ranking, SIP projections, and interactive business intelligence dashboards.
## 🚀 Project Objectives

- Analyze historical mutual fund performance.
- Compare category-wise returns and fund rankings.
- Identify top-performing Asset Management Companies (AMCs).
- Evaluate risk-adjusted returns and expense efficiency.
- Build SIP growth projections and investment insights.
- Create interactive dashboards for data-driven decision making.

## 📂 Project Workflow

### 1. Data Collection
- Collected Indian Mutual Fund performance dataset.

### 2. Data Cleaning & Preprocessing
- Handled missing values and duplicates.
- Converted data types and standardized text columns.
- Applied median imputation for numerical features.
- Created analysis-ready datasets.

### 3. Feature Engineering
Generated custom investment metrics:

- Risk Adjusted Return
- Expense Efficiency
- Return Consistency
- Affordable SIP Classification
- Return Score
- Alpha-Beta Score
- Final Composite Score
- Fund Ranking

### 4. Exploratory Data Analysis (EDA)
Performed analysis on:
- Category-wise average returns
- Expense ratio vs return relationship
- Correlation between fund metrics
- Top-performing funds
- Return consistency and risk analysis

### 5. SQL Analysis
Implemented SQL queries for:
- Best Performing AMCs
- Top Mutual Funds
- Category-wise Return Analysis
- Beginner-Friendly Funds
- Highest and Lowest Performing Funds
- Sub-category Analysis

### 6. Excel Analysis
- SIP calculations
- Financial formulas
- Investment projections
- Dashboard calculations

### 7. Power BI Dashboard
Developed interactive dashboards featuring:
- Market Overview Dashboard
- Top 30 Funds Dashboard
- SIP Growth Calculator
- Fund Performance Dashboard
- Investment Insights Dashboard

---

## 📈 Key Insights

- Equity funds showed higher long-term returns.
- Some AMCs consistently outperformed others.
- Risk-adjusted metrics provided better investment evaluation.
- SIP investments demonstrated significant compounding benefits.
